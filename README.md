Здравствуйте всем. Это мой проект. В этом проекте будет установка расширения Visual Studio Code для работы с микроконтроллерами (в моём случае пока что только для ESP8266), а также с помощью кода сделаем светодиод с миганием. Мой проект делится на несколько этапов.

Этапы:

1. Скачиваете VScode на официальном сайте и после загрузки включаете его.

2. Далее слева будут вкладки, где будет отображена кнопка с колбочкой под названием Extensions (Расширение).

3. Нажимаете её и в поиске ищете PlatformIO.

4. В расширении скачиваете его и ждёте конца загрузки.

5. Потом ниже появится кнопка с лицом муравья с надписью PlatformIO, и нажимаете на неё.

6. Дальше откроется меню, или нажимаете на Open в Quick Access слева от центра.

7. Нажимаете в меню New Project.

8. Открывается меню с выбором имени проекта, модели платы (board = d1_mini, board вписывается вручную потом, а пока выбираете Espressif ESP8266 ESP-12E) и framework (Arduino по умолчанию у меня ставилось).

9. Нажимаете finish, и проект создан.

10. Далее берёте микроконтроллер и подключаете его через USB-кабель.

11. Проверяете в диспетчере устройств свой порт (COM). Если он не отображается или при дальнейшей прошивке не работает, то установите драйвер к микроконтроллеру (удалите его и скачайте по этой ссылке https://sparks.gogo.co.nz/assets/_site_/downloads/CH34x_Install_Windows_v3_4.zip, и при нажатии на кнопочку install всё это установится, и всё готово).

12. В папке src пишете код в main.cpp, а в platform.ini можете кастомизировать его по своему, но обязательно надо будет прописать upload_port = [Свой Порт], чтобы при прошивке он увидел именно наш порт.

13. Далее нажимаете снизу или в правом верхнем углу на галочку (build), которая компилирует код.

14. В конце прошиваете микроконтроллер при нажатии на кнопочку со стрелочкой.

По итогу у вас будет успешно работать микроконтроллер с тем функционалом, который был прописан в src main.cpp.



***Дополнительно: Микроконтроллер понимает состояние, а именно "Цифровое" - есть ли напряжение(HIGH или наоборот в зависимости от платы) или его нету(LOW или наоборот в зависимости от платы).
